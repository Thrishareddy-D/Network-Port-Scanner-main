# Network Port Scanner using Python

## 📌 Project Description

This project is a Python-based Network Port Scanner that scans a target IP address to identify OPEN and CLOSED TCP ports. It also displays the common network service associated with well-known ports.

---

## 🚀 Features

- Scan any IP Address
- Scan a custom port range
- Detect OPEN and CLOSED ports
- Display common service names (HTTP, HTTPS, SSH, FTP, etc.)
- Show "Unknown Service" for undefined ports
- Uses Python Socket Programming

---

## 🛠 Technologies Used

- Python 3
- Socket Module
- Visual Studio Code

---

## ▶️ How to Run

```bash
python port_scanner.py
```

Example:

```
Enter IP Address: 127.0.0.1
Enter Start Port: 80
Enter End Port: 82
```

Output:

```
Port 80 is OPEN (HTTP)
Port 81 is CLOSED (Unknown Service)
Port 82 is CLOSED (Unknown Service)
```
